Proxmox Server v1.0

Self-hosted infrastructure build • $200 eBay build

I wanted to stop paying for services I could run myself.
What started as a simple media server turned into a deep dive into self-hosted infrastructure.

TLDR

Built a multi-service homelab on a $200 Lenovo M720s running Proxmox. Hosts Jellyfin, media automation, Nginx reverse proxy, Cloudflare Zero Trust tunnels, and a self-hosted AI stack (Ollama/N8N) across 6 isolated VMs/containers.

Hardware & Operating Systems

Hardware

Chassis: Lenovo ThinkCentre M720s Small Form Factor (SFF)

CPU: Intel i5-8400 (6 cores, QuickSync transcoding)

RAM: 24GB DDR4 (mixed sticks)

Boot Storage: 512GB NVMe SSD + heatsink

Media Storage: 8TB Seagate IronWolf HDD

Backup: 8TB External HDD

Operating Systems

  • Proxmox VE — Bare-metal hypervisor
  • Ubuntu — LXC containers
  • Ubuntu Server — Virtual Machines
  • Docker — Container orchestration inside VMs

Functionality

ID Type OS Application(s) Purpose
100 'media' LXC Ubuntu n/a Dedicated media storage

Other Notable Services

  • Samba File Share
  • SSH with key auth + passphrase (root disabled)
  • Cloudflare Zero Trust Tunnels
