TLDR

I gained initial access to the Pickle Rick web server by brute-forcing directories, finding the login credentials in robots.txt, and logging into the command panel. I then located the first ingredient by directly accessing a hidden file, found the second ingredient in Rick’s home directory (bypassing disabled readers with the strings command), and retrieved the third ingredient after discovering full sudo privileges as www-data and accessing the root directory.

Challenge Description

This beginner-friendly room is themed around Rick and Morty. The goal is to exploit a vulnerable web server to locate three secret ingredients needed to turn Pickle Rick back into a human.

Reconnaissance

To get started, I verified connectivity and checked for open ports.

While I was in the CLI, I pulled the page's source code and found our username: R1ckRul3s

curl -v http://10.64.147.126

Since this is a web server, I ran GoBuster to see what other pages were available.

Directory Brute Forcing

Found robots.txt which contained our (suspected) password: Wubbalubbadubdub

Also located our login page (login.php) and logged in.

After logging in, I was brought to this peusdo-CLI where I could run basic commands. First ingredient was located in the current directory.

Found the second ingredient under the user directory '/home/rick/second_ingredients'

strings '/home/rick/second ingredients'

I'll admit, I was a bit stumped at first - so I checked my privileges and found I had full system privileges. Checked the root directory and found our final ingredient

sudo -l
